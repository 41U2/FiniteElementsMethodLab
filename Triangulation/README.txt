Модуль, в котором реализована триангуляция прямоугольной области, заданная следующими параметрами: 
1. Начальная точка (левый нижний угол)
2. Число точек разбиения по х и по у
3. Шаги разбиения

На выход подать:
1. Массив вершин
2. Массив смежностей вершин (std::vector<std::vector<int>>, где каждой вершине осответствует набор смежных с ней вершин)

Для вершины будет удобно создать класс Vector2D, хранящий х и у координаты
